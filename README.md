# Restaurante Comeu Morreu

Sistema web desenvolvido como projeto acadêmico da disciplina de Modelagem de Dados para Web.

## Descrição

O projeto simula o funcionamento de um restaurante, permitindo a interação entre clientes e equipe por meio de páginas web desenvolvidas com HTML, CSS e JavaScript.

## Funcionalidades

* Login de usuários
* Cadastro de clientes
* Visualização do cardápio
* Reserva de mesas
* Área destinada ao garçom
* Interface responsiva para navegação

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

## Estrutura do Projeto

```text
ProjetoWeb/
├── css/
├── img/
├── js/
├── index.html
└── login.html
```

## Como Executar

1. Faça o download ou clone o repositório.
2. Abra o arquivo `index.html` em um navegador.
3. Navegue pelas funcionalidades do sistema.

## Objetivo do Projeto

Aplicar conceitos de desenvolvimento web, organização de interfaces, estilização com CSS e manipulação de comportamento com JavaScript.
